# Navigation Puck Addon

![image](./navigation_puck_addon_icon.png)

Navigation Puck adds a compact viewport navigation menu to Blender. It is inspired by Sketchbook Pro's navigation puck and gives quick access to orbit, pan, zoom, camera flip, and related view actions from the current editor context.

## Requirements

- Blender 4.2 or newer

## Install

1. Download the release zip for the addon.
2. In Blender, open `Edit > Preferences > Get Extensions`.
3. Use `Install from Disk` and choose the zip file.
4. Enable `Navigation Puck Addon`.

## Activation

The default activation mode is `Hotkey`. Press and hold `Left Alt` to open the Navigation Puck menu, then drag toward an action and release. The hotkey can be changed in the add-on preferences.

Preferences also include two alternate activation modes:

- `Shortcut Button`: shows a small shortcut icon near the cursor that fades in as the pointer approaches it.
- `Menu Near Cursor`: opens the full puck menu near the cursor without the small shortcut button.

The shortcut placement, button sizes, menu sizes, drag-select threshold, and debug bounds can be adjusted from the add-on preferences.
